# 🏠 California House Price Predictor

## Overview
A machine learning web app that predicts median house prices in California based on location and property features.

## Model Performance
- **R² Score**: 0.81 (Explains 81% of price variation)
- **Mean Absolute Error**: $33,000
- **Algorithm**: Random Forest Regressor

## Features Used
- Median Income
- House Age
- Average Rooms & Bedrooms
- Population & Occupancy
- Latitude & Longitude
